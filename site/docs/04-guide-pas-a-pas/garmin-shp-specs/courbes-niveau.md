# Courbes de niveau

Les courbes de niveau sont générées à partir du **RGE ALTI** (Référentiel à Grande Échelle
Altimétrique) de l'IGN.

!!! note "En cours d'intégration"
    L'intégration des courbes de niveau dans le pipeline automatisé est en cours.
    Pour l'instant, les courbes de niveau ne sont pas incluses dans les cartes téléchargeables.
