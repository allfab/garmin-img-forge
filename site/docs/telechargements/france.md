# :material-map: France métropolitaine

!!! tip "Choisir la couverture"
    - **France entière (FXX)** : un seul fichier pour toute la métropole — utile si
      vous avez une carte SD de grande capacité.
    - **Moitiés NORD / SUD** : deux fichiers plus légers, pratiques si votre GPS
      limite la taille `gmapsupp.img`.
    - **Quadrants Garmin (NE / NO / SE / SO)** : calqués sur le découpage de la
      *Garmin TOPO France v7 PRO*, idéal pour les appareils à mémoire limitée.

## Quadrants Garmin (TOPO France v7 PRO)

officielles *Garmin TOPO France v7 PRO*. L'Île-de-France est partagée entre
les quadrants Nord-Est et Nord-Ouest, conformément au choix Garmin.
