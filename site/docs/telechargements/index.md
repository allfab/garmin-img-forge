# Téléchargements des cartes Garmin

Les cartes sont distribuées au format **`gmapsupp.img`**, le format d'installation universel
pour les GPS Garmin.

## Prérequis

!!! info "Compatibilité GPS"
    Ces cartes sont compatibles avec tous les GPS Garmin supportant les cartes installées :
    fenix, Oregon, eTrex, Edge (certains modèles), Montana, GPSMAP, Alpha, Astro...

!!! warning "Installation"
    Copiez le fichier `gmapsupp.img` à la racine de votre carte SD ou dans le dossier `Garmin/`
    de votre GPS. Rebaptisez-le si nécessaire pour éviter les conflits avec d'autres cartes.

## Disponibilité

Les cartes sont générées périodiquement depuis la BD TOPO IGN. Les téléchargements correspondent
à la dernière version disponible.
