# Garmin Map Downloads

Maps are distributed in **`gmapsupp.img`** format, the universal installation format
for Garmin GPS devices.

## Prerequisites

!!! info "GPS Compatibility"
    These maps are compatible with all Garmin GPS devices supporting installed maps:
    fenix, Oregon, eTrex, Edge (certain models), Montana, GPSMAP, Alpha, Astro...

!!! warning "Installation"
    Copy the `gmapsupp.img` file to the root of your SD card or in the `Garmin/`
    folder of your GPS. Rename it if necessary to avoid conflicts with other maps.

## Availability

Maps are generated periodically from IGN BD TOPO data. Downloads correspond
to the latest available version.
