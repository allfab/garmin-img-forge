# Support the project

**GARMIN IMG FORGE** is an open source project, free and maintained in its author's spare time. If the maps are useful to you, you can contribute to its sustainability.

---

## Why support?

Producing and maintaining quality Garmin topographic maps requires a real investment:

- **Development** — Hundreds of hours of work to create and evolve the pipeline tools: [ogr-polishmap](../the-project/ogr-polishmap.md), [mpforge](../the-project/mpforge.md), [imgforge](../the-project/imgforge.md)
- **Server infrastructure** — Site hosting, map generation pipeline, download file storage
- **Ongoing maintenance** — Tracking IGN BD TOPO updates, compatibility with new Garmin GPS devices, bug fixes
- **Hardware** — Garmin GPS devices needed to test maps in real conditions

## Other ways to contribute

Financial support is not the only way to help the project:

- **Report a bug** — Open an [issue](https://github.com/allfab/garmin-img-forge/issues) if you encounter a problem with a map or the site
- **Contribute to the code** — The [GitHub repository](https://github.com/allfab/garmin-img-forge) is open to contributions
- **Share the project** — Tell others about it, on hiking, mountain biking or cartography forums

Thank you for your support, in whatever form it takes.
